# Contributing to OpenSource Sustainability Intelligence (OSSI)

Thanks for your interest in contributing! We welcome all kinds of contributions: code, documentation, design, testing, and ideas.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Create an issue** to discuss major changes before submitting a PR.
3. **Keep commits clear and focused** with descriptive messages.
4. **Write clean, readable code** and add comments where necessary.
5. **Update documentation** when you add or change features.
6. **Open a pull request** with a clear summary of changes.

## Development Setup (Planned)

The development setup will be documented once the initial MVP structure is added.

## Reporting Issues

If you find a bug or want a new feature, please open an issue and include:
- Steps to reproduce (if applicable)
- Expected vs actual behavior
- Screenshots or logs if helpful

## Code of Conduct

By participating, you agree to follow our Code of Conduct.
